
# TaNA's Portfolio

My Portfolio made with Vue.js

## Description

This Portfolio is build with Vue.js + Webpack + HTML5 + CSS3

[TaNA's Portfolio URL](https://machio77777.github.io/vue-portfolio/)

## Development

```
npm install -g vue-cli
vue init webpack プロジェクト名
npm run dev
npm run build
```

## Technology

- vue.js
- vue-cli
- Webpack
- HTML5
- CSS3
